# Fuck Facebook's Feed

A minimal browser extension that removes the Facebook feed. It works on both
Chrome and Firefox.

## Installation

### Firefox

Download the latest signed `.xpi` from the
[releases page](https://github.com/mahyarmirrashed/fuck-facebook-feed/releases)
and install it via `about:addons` → gear icon → **Install Add-on From File...**.

### Chrome

Download the latest `.zip` from the
[releases page](https://github.com/mahyarmirrashed/fuck-facebook-feed/releases),
extract it, then load it via `chrome://extensions` with **Developer Mode**
enabled → **Load unpacked**.
